## Why a portfolio?

I've been building automations and tooling quietly for a while now. Scripts that run at 3am, integrations nobody asked for but everyone needed, and Enfocus Switch flows that turned 2-hour manual jobs into a coffee break.

At some point, it felt right to have a place to point people to.

## What you'll find here

- **Project writeups** — How I built things, what broke, and what I'd do differently.
- **Automation tips** — Practical n8n, Switch, and API patterns I use regularly.
- **Dev notes** — Short posts on things I figured out and didn't want to forget.

## About the stack

This site runs on Next.js 15 with the App Router, Tailwind CSS v4, and Framer Motion for animations. Blog posts are just Markdown files in `content/blog/` — no CMS, no database, no fuss. The GitHub projects page pulls live from the GitHub API.

Simple. That's the point.
